# instalar

- Primer paso: Crear variables de entorno

    ```bash
    Copiar .env.sample a .env
    ```

- Segundo paso: instalar node modules
    ```bash
    npm install
    ```
- Tercer Paso: Ejecutar modo desarollo
    ```bash
    npm run dev
    ```
- Cuarto Paso: Ejecutar modo desarollo
    ```bash
    npm run start
    ```
